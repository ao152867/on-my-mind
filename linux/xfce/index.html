XFCE

Backup panel configuration

Open up the Xfce Panel Switch:

Once you’ve customized your panel to your liking select Save configuration it is the button to the right of the gears near the bottom left. Give it a name and then select it and click Export (fourth icon).

To import it, click the Import button (fifth icon) then Apply Configuration (first icon; the gears).

Keyboard settings

Application Shortcuts

Once at the keyboard settings, click on Application Shortcuts.

Change these commands’ shortcuts to:

command shortcut
xfce4-appfinder Ctrl+Space or Super+Space
xfce4-terminal Super+t
Thunar Super+f
exo-open –launch WebBrowser Super+w

Keyboard shortcuts to add:

command shortcut
xflock4 Super+L
xfce4-screenshooter -f -m -d 5 Super+5
atom Super+a
VirtualBox Super+v
xfce4-screenshooter -w Alt+Print (is this already there ?)

Now there are three shortcuts to lock the screen (xflock4) the other two are Ctrl+Alt+Delete and Ctrl+Alt+l

The screenshot command takes a screenshot of the whole screen -f, captures the mouse cursor -m and waits 5 seconds before taking it. -d 5

Layout

Unselect Use system defaults then under Change layout option: Alt+Caps Lock.

Window Manager Tweaks

Menu → Settings → Window Manager Tweaks → Accessibility tab

To be able to scroll in a non-focused window without changing focus: Uncheck Raise windows when any mouse button is pressed.

Questionable change

Change Key used to grab and move windows: to Super

Panel

My preference is one panel so delete any other panels if they exist. I prefer the traditional looking panels too so if one looks like the macOS bar get rid of that one.

Questionable change

Set Window grouping: to Always. This has the benefit of showing the title bar of all windows of the same type when you click on its icon on the bar instead of having to hover over each one to see which is which.

Clock

Right-click on the clock on the panel and select Properties. Set the Format: to Custom Format and set it to this format: <b>%T // %F // %a</b>
